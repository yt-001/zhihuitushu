# 书籍管理接口文档

## 概述

本文档描述智慧图书阅读平台管理端的书籍管理接口，包括书籍的增删改查（CRUD）操作。

- **基础路径**: `/api/admin/books`
- **认证方式**: JWT Token（需管理员权限）
- **响应格式**: JSON

## 统一响应格式

所有接口返回统一的响应格式：

```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```

- `code`: 状态码，200表示成功，其他表示错误
- `message`: 提示信息
- `data`: 返回数据

## 1. 创建书籍

### 基本信息
- **接口地址**: `/api/admin/books`
- **请求方法**: `POST`
- **Content-Type**: `application/json`

### 请求参数

| 字段名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| title | String | 是 | 书籍标题 |
| author | String | 是 | 作者 |
| cover | String | 否 | 封面图片URL |
| categoryId | Long | 是 | 分类ID |
| publisher | String | 否 | 出版社 |
| publishDate | LocalDate | 否 | 出版日期（格式：yyyy-MM-dd） |
| language | String | 否 | 语言 |
| pages | Integer | 否 | 页数 |
| isbn | String | 否 | ISBN编号 |
| description | String | 否 | 书籍描述 |
| status | Integer | 否 | 状态：0-下架，1-上架（默认为1） |

### 请求示例

```json
{
    "title": "Java编程思想",
    "author": "Bruce Eckel",
    "cover": "https://example.com/cover.jpg",
    "categoryId": 1,
    "publisher": "机械工业出版社",
    "publishDate": "2023-06-15",
    "language": "中文",
    "pages": 800,
    "isbn": "978-7-111-23456-7",
    "description": "一本经典的Java编程入门书籍",
    "status": 1
}
```

### 响应示例

```json
{
    "code": 200,
    "message": "创建成功",
    "data": {
        "id": 101,
        "title": "Java编程思想",
        "author": "Bruce Eckel",
        "cover": "https://example.com/cover.jpg",
        "categoryId": 1,
        "categoryName": "编程技术",
        "publisher": "机械工业出版社",
        "publishDate": "2023-06-15",
        "language": "中文",
        "pages": 800,
        "isbn": "978-7-111-23456-7",
        "description": "一本经典的Java编程入门书籍",
        "rating": 0.0,
        "reviewCount": 0,
        "readCount": 0,
        "downloadCount": 0,
        "status": 1,
        "createdAt": "2024-01-15T10:30:00",
        "updatedAt": "2024-01-15T10:30:00"
    }
}

## 5. 分页查询书籍列表

### 基本信息
- **接口地址**: `/api/admin/books`
- **请求方法**: `GET`

### 请求参数（Query String）

| 字段名 | 类型 | 是否必填 | 默认值 | 说明 |
|--------|------|----------|--------|------|
| page | Integer | 否 | 1 | 当前页码 |
| size | Integer | 否 | 10 | 每页数量 |
| keyword | String | 否 | - | 关键词（搜索标题或作者） |
| categoryId | Long | 否 | - | 分类ID |
| status | Integer | 否 | - | 状态筛选 |
| language | String | 否 | - | 语言筛选 |
| publisher | String | 否 | - | 出版社筛选 |
| sortBy | String | 否 | createdAt | 排序字段（createdAt/readCount/rating/publishDate） |
| sortOrder | String | 否 | desc | 排序方向（asc/desc） |

```
GET /api/admin/books?page=1&size=10&keyword=Java&categoryId=1&status=1&sortBy=readCount&sortOrder=desc

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录/认证失败 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
